# 🌱 AgroNova — Futuro Sustentável

Projeto de site moderno sobre agronegócio sustentável, tecnologia no campo e inovação agrícola.

---

# 📖 Sobre o Projeto

O **AgroNova** foi desenvolvido com o objetivo de apresentar um conceito moderno de agronegócio sustentável, unindo:

- 🌿 Sustentabilidade
- 🚜 Tecnologia agrícola
- 💧 Economia de recursos
- ☀️ Energia renovável
- 📈 Agricultura inteligente

O site possui design moderno, responsivo e interativo.

---

# 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

---

# 🎨 Funcionalidades

✅ Menu de navegação fixo  
✅ Layout responsivo  
✅ Hero section moderna  
✅ Cards interativos  
✅ Estatísticas animadas  
✅ Botão com interação JavaScript  
✅ Formulário de contato  
✅ Efeitos hover e animações
